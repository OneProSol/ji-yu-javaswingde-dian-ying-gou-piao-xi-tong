### 一、相关技术
- 前端：`javaswing`。
- 后端：`java`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 登录、2. 主页、3. 热门电影、4. 电影详情、5. 电影场次、6. 电影选座购票、7. 我的电影、8. 查找电影、9. 科幻电影、10. 动作电影、11. 爱情电影

管理员功能：1. 登录、2. 电影添加、3. 电影维护、4. 影院维护、5. 场厅维护、6. 场次添加、7. 场次维护、8. 影票管理
